RollerBall

  • RollerBall est un jeu Android initialement développé par deux étudiants, de l'Iut Informatique de Clermont-Ferrand, lors d'un projet de 4 semaines.

Manuel de l’utilisateur :

  • Le but du jeu est de rentrer la balle dans le bon trou.
  • Une multitude de niveau vous attend prochainement. Pour réussir un niveau, il faut arriver dans le bon trou (celui avec le cercle vert).
  • Pour déplacer la boule, vous devez incliner votre téléphone dans un sens.
  • Le jeu est jouable en mode portrait, comme en mode paysage. Et vous aurez par ailleurs intérêt à changer d’orientation dans certains niveaux, car ils seront plus faciles à réussir dans une certaine orientation.
  • A chaque fois que vous terminez un niveau, vous débloquez le niveau suivant. Pour le moment seul un nombre limité de niveau sont disponibles.
  • Le jeu possède une fonctionnalité de partage des scores sur le Google Play Games. Pour utiliser cette fonctionnalité, vous devez vous connecter à l’aide du bouton se trouvant sur le menu de démarrage. Une fois connecté, votre score sera partagé automatiquement à chaque fin de niveau.
